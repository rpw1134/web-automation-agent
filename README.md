Steps for getting started (writing here so I don't forget when I'm done!)
* Run 'poetry install'
* Properly configure your python interpreter using the path shown when running 'poetry env info --path'
* Run 'poetry run playwright install'

This project will include the following:
1. Agent tool call loops
2. Error handling
3. Playwright usage to manage the browser
4. Interface for managing the agent
5. Potentially other features
